## 🎯 使用示例

### 关键词触发
```
用户: 独角兽
机器人: [正常回复]
```

### 无关键词（第1次）
```
用户: 今天天气真好
[AI 判断能否接话]
→ 能接话：正常回复
→ 不能接话：静默
```

### 无关键词（第2-3次）
```
用户: 在干嘛
[进入待机状态，不回复]
```

### 待机状态
```
用户: 随便说什么
[忽略，不回复]

[30秒后]
机器人: 哥哥...如果想和独角兽说话，记得叫我的名字哦...
```

### 关键词激活
```
用户: 独角兽
[退出待机，正常对话]
机器人: 哥哥叫我了...有什么事吗...的说
```

## 🐛 已知问题

1. **启动日志不完整**
   - 原因：系统初始化在 login 事件中，但日志可能被覆盖
   - 影响：不影响功能，只是日志显示不全
   - 解决：已添加错误处理和降级机制

2. **语音合成失败**
   - 原因：Flask 服务未启动
   - 影响：无语音回复，文字回复正常
   - 解决：启动 `python app.py` 或忽略

3. **Claude 未配置**
   - 原因：缺少 ANTHROPIC_API_KEY
   - 影响：使用 Gemini 备用服务
   - 解决：添加环境变量或继续使用备用服务
