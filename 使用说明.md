# 独角兽微信机器人 - 快速启动指南

## 📦 一键启动脚本

### 1. 启动独角兽.bat
**功能：** 一键启动所有服务
- 自动检查 Python 和 Node.js 环境
- 启动 Flask 后端服务（端口 5000）
- 启动微信机器人客户端
- 打开两个独立窗口，互不干扰

**使用方法：**
```
双击 "启动独角兽.bat"
```

---

### 2. 停止独角兽.bat
**功能：** 一键停止所有服务
- 停止 Flask 后端
- 停止微信机器人
- 清理所有相关进程

**使用方法：**
```
双击 "停止独角兽.bat"
```

---

### 3. 检查状态.bat
**功能：** 检查系统状态
- Python 环境
- Node.js 环境
- Flask 服务运行状态
- 依赖库安装情况
- 关键文件完整性

**使用方法：**
```
双击 "检查状态.bat"
```

---

## 🚀 首次使用流程

### 步骤 1：安装依赖
```bash
# Python 依赖
pip install -r requirements.txt

# Node.js 依赖
cd src
npm install
```

### 步骤 2：启动服务
```
双击 "启动独角兽.bat"
```

### 步骤 3：扫码登录
在微信机器人窗口中扫描二维码登录

## ⚙️ 配置说明

### 群消息配置
编辑 `src/bot-unicorn.js`：

```javascript
const GROUP_CONFIG = {
  keywords: ['独角兽', '妹妹'],  // 触发关键词
  cooldownMs: 5000,              // 群聊冷却时间（毫秒）
  adminIds: []                   // 管理员微信ID（无冷却限制）
}
```

**添加管理员示例：**
```javascript
adminIds: ['wxid_abc123', 'wxid_def456']
```

### 模型配置
编辑 `wechat_bot_integrated.py`：

```python
bot = WeChatUnicornBot()
bot.scheduler = UnicornScheduler(
    deepseek_api_key=DEEPSEEK_API_KEY,
    deepseek_base_url=DEEPSEEK_BASE_URL,
    gpt_model="GPT_weights_v2Pro/xxx-e50.ckpt",  # GPT 模型路径
    sovits_model=""                               # SoVITS 模型路径
)
```

## 🔧 常见问题

### Q1: Flask 服务启动失败
**解决方法：**
1. 运行 `检查状态.bat` 查看依赖
2. 确保端口 5000 未被占用
3. 检查 Python 版本 >= 3.8

### Q2: 微信机器人无法登录
**解决方法：**
1. 确保 Flask 服务已启动
2. 检查网络连接
3. 尝试重启机器人

### Q3: 群聊不响应
**解决方法：**
1. 确认消息包含关键词（"独角兽"或"妹妹"）
2. 或者 @ 机器人
3. 检查是否在 5 秒冷却期内

### Q4: 语音发送失败
**解决方法：**
1. 确保安装了 `pilk`：`pip install pilk`
2. 检查 TTS 服务是否正常
3. 查看 Flask 日志排查错误
